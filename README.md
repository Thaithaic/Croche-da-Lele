# 🧶 Crochê da Lélé

Site oficial do Crochê da Lélé — amigurumis artesanais feitos à mão com amor.

## 📁 Estrutura

```
crochet-da-lele/
├── index.html   ← site completo (HTML + CSS + JS em um arquivo só)
└── README.md    ← este arquivo
```

## 🚀 Como rodar localmente

Basta abrir o arquivo `index.html` no navegador. Não precisa de servidor.

## 🌐 Deploy no GitHub Pages

1. Sobe os arquivos neste repositório
2. Vai em **Settings → Pages**
3. Em **Source**, seleciona `main` branch e salva
4. O site fica disponível em: `https://seu-usuario.github.io/crochet-da-lele/`

## ✏️ Como editar

Abra o `index.html` em qualquer editor de texto (Bloco de Notas, VS Code, etc).

- **Textos** → pesquisa pelo trecho e edita direto
- **Cores** → no topo do CSS, seção `:root { }`, estão todas as variáveis de cor
- **Preços** → pesquisa por `R$` e atualiza o valor
- **Links** → pesquisa por `wa.me` (WhatsApp) e `instagram.com` para atualizar
